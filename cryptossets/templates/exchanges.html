What is Cryptossets?

Cryptossets is a project done by Hossein Sokhandan in ProgrammingSystems course taken in 2021. This project is open-source and you can explore its code in github in this repo. Cryptossets will get your api keys of popular exchanges and give you a summary of what you have been buying/selling so far. The value of Cryptossets is that you can measure your trades, "you can progress what you don't measure.".
